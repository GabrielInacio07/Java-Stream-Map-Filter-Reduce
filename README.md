# ☕ Java – Map, Filter and Reduce (Stream API)

Este repositório contém exemplos práticos utilizando **Map, Filter e Reduce em Java**, aplicados através da **Stream API**.

O objetivo deste projeto é demonstrar como manipular coleções de dados de forma mais **funcional, expressiva e eficiente**, substituindo loops tradicionais por pipelines de processamento.

---

## 📚 Conceitos aplicados

Neste projeto são explorados conceitos importantes da **programação funcional em Java**, incluindo:

* 🌊 **Stream API**
* 🔁 Processamento de coleções com streams
* 🔧 **map()** – transformação de dados
* 🔎 **filter()** – filtragem de elementos
* ➕ **reduce()** – agregação de valores
* 🔗 Encadeamento de operações (pipeline)

Essas operações permitem processar sequências de dados de maneira declarativa, evitando loops explícitos e melhorando a legibilidade do código.
